# Endometriosis Awareness Website 🎗️

An open-source website dedicated to endometriosis awareness, education, and advocacy.

**🌐 Live Site:** [View Website](https://YOUR_USERNAME.github.io/endometriosis-website/)

## About

Endometriosis affects approximately **190 million people** worldwide — yet it takes an average of **7–10 years** to receive a diagnosis. This site aims to:

- 📖 **Educate** — Clear, accurate information about endometriosis
- 🔍 **Help recognize symptoms** — Know the signs and when to seek help
- 💛 **Support** — Real stories, resources, and community links
- 📢 **Advocate** — Actionable ways to make a difference

## Features

- Fully responsive design (mobile-first)
- Pure HTML, CSS, and JavaScript — no frameworks or dependencies
- Accessible (semantic HTML, focus states, reduced motion support)
- Fast loading — no build step required
- Hosted free on GitHub Pages

## Deployment

This site is designed for **GitHub Pages**:

1. Push to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, root `/`
4. Your site will be live at `https://username.github.io/repo-name/`

## Development

No build tools required. Just open `index.html` in a browser:

```bash
open index.html
# or use a local server:
python3 -m http.server 8000
```
